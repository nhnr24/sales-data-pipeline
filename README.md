# Sales Data Pipeline

## Overview

This project implements an end-to-end ETL pipeline using Python and performs data processing using PySpark to generate business insights from sales data.

## Features

* Data extraction from CSV files
* Data cleaning and transformation using Python
* Data processing using PySpark (filter, groupBy, aggregation)
* Data visualization using bar charts

## Key Insights

* Electronics category generated the highest revenue
* South region shows strong and consistent performance
* Fashion category has higher quantity but lower revenue

## Project Workflow

1. Data Extraction from CSV
2. Data Cleaning and Transformation (Python)
3. Data Processing (PySpark)
4. Data Analysis and Visualization

## Tools Used

Python, Pandas, PySpark, Matplotlib

1. Run ETL Pipeline:

2. Run PySpark Analysis:

3. Run Visualization:
